## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-a-v1.jar
```

## Задание
Напишите регулярное выражение, проверяющее является ли строка адресом эл. почты. Реализуйте приложение демонстрирующее работоспособность регулярного выражения.
