## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-b-v1.jar
```

## Задание
Реализуйте алгоритм расчета энтропии по Шеннону. Вывод ограничте до двух знаков после запятой. Алгоритм расчета выделите в отдельный метод.
