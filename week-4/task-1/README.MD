## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-1-v1.jar
```

## Задание
Напишите программу, которая открывает и выводит на экран содержимое текстового файла. 
При возникновении ошибок, выводите сообщения на экран в соответствии с возникшей ошибкой.
