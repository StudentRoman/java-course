## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-3-1.0-SNAPSHOT.jar
```

## Задание
Напишите программу, которая на вход получает длину ребра додекаэдра и выводит его объем.
