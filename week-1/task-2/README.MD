## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-2-1.0-SNAPSHOT.jar
```

## Задание
Вывести максимальные и минимальные значения числовых типов данных и обосновать почему они таковы (в виде комментариев в коде).
