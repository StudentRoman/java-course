## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-1-1.0-SNAPSHOT.jar
```

## Задание
Написать программу, выводящую фразу “Привет, user!”, где user - имя пользователя, вводимое пользователем.
