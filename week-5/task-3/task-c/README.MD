## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-c-v1.jar
```

## Задание
На вход программы подается строка, содержащая только символы '(', ')', '{', '}', '[' и ']', определите, валидна ли входная строка.
