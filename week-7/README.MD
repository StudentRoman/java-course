## Запуск программы

### Maven

Для сборки проекта:

```sh
mvn clean compile assembly:single
```

Для запуска проекта:

```sh
java -jar .\target\week-7-v1-jar-with-dependencies.jar
```

### Gradle

Для сборки проекта:

```sh
.\gradlew clean build
```

Для запуска проекта:

```sh
java -jar .\build\libs\week-7-v1.jar
```
