## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-a-v1.jar
```

## Задание
Напишите программу, которая на вход получает набор чисел в виде массива. Необходимо вывести количество чисел, равных максимальному.
