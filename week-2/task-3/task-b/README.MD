## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-b-v1.jar
```

## Задание
Напишите программу, реализующую алгоритм сортировки MergeSort. Сортировку оформить отдельным методом.
