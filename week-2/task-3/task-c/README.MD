## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-c-v1.jar
```

## Задание
Напишите программу, которая на вход получает набор чисел в виде массива. Необходимо найти медиану этого набора чисел.
