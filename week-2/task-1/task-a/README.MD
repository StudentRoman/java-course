## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-a-v1.jar
```

## Задание
Напишите программу, которая принимает на вход целое число - возраст и выводит его с припиской год/лет и правильно склоняет (например - 21 год/22 года, 18 лет и т.д.)
