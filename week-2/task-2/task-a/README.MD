## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-a-v1.jar
```

## Задание
Напишите программу, которая на вход получает высоту пирамиды и выводит ее псевдосимволами в консоль.
