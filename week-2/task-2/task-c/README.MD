## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-c-v1.jar
```

## Задание
Напишите программу, реализующую алгоритм определения простых чисел (решето Эратосфена).
