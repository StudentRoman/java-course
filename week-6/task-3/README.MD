## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-3-v1.jar
```

## Задание
Напишите программу для поиска файлов по названию или содержимому (для текстовых файлов) в выбранной директории.
