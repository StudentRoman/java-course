## Запуск программы

Для сборки проекта:

```sh
mvn package
```

Для запуска проекта:

```sh
java -jar .\target\task-1-v1.jar
```

## Задание
Необходимо написать метод, который принимает на вход строку s и выводит пару двух элементов: Самый часто встречающийся символ и Количество его повторов
